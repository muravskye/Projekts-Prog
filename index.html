Welcome to the Meal Data Analyzer

Use the navigation bar to upload data or view analysis.

Note: This is a simplified application for demonstration.

  • Upload a CSV file (using the provided format) on the 'Upload CSV' page. This will load the data into the database.
  • 'View CSV Data & Plot' shows data directly from the last uploaded CSV file and a histogram.
  • 'View Database Data' shows the data currently stored in the SQLite database.
